OpenAI 내부자가 직접 알려주는 GPT 5.6과 Codex 활용법

OpenAI 내부자가 직접 알려주는 GPT-5.6과 Codex 활용법

26.07.23.069 min read

요약

GPT 보이스(Live)와 실시간 통번역 데모

조코딩이 오픈AI의 가브리엘 추아를 다시 초대해 GPT-5.6과 Codex의 새 기능을 데모로 소개한다. 먼저 이날 출시된 GPT 보이스를 활용해 별도 통역사 없이 에어팟과 챗GPT 보이스 모드 계정만으로 실시간 통번역을 구현하는데, ‘한국어가 들리면 영어로 번역하고 영어는 무시하라’는 식으로 지시하면 말과 거의 동시에 번역이 나와 실시간 통역기가 된다. 반대 방향(영어→한국어)도 동시에 세팅해 양방향 통역이 가능하다. 기술적으로는 두 가지 혁신이 있는데, 예전에는 음성을 텍스트로 바꾸고 LLM이 처리한 뒤 다시 음성으로 바꾸는 단계별 방식이라 느렸고 GPT-4o는 턴 기반이라 대화가 부자연스러웠던 반면, 이제 ‘GPT-Live-1’이 말하는 동안 바로 응답해 끊김 없는 상호작용이 가능하다는 것이다. 또 더 높은 지능이 필요할 때는 ‘위임(delegation)’ 기능으로 GPT-5.5 같은 모델에 작업을 넘겨(예: 검색) 처리 결과를 다시 가져온다.

배포 범위와 Cerebras 기반 GPT-5.6 Sol

가브리엘 추아는 이 기능을 무료 버전을 포함해 Go·Plus·Pro 전부에서 쓸 수 있고 미니 버전(GPT-Live-1-Mini)도 있으며 API도 함께 제공한다고 밝힌다. 나아가 Cerebras 팀과 협력해 GPT-5.6 Sol을 Cerebras에서 선보이는데, Cerebras는 추론(인퍼런스)이 매우 빠른 칩을 만드는 곳으로 초당 최대 750토큰까지 처리해 프론티어급 AI 성능을 전례 없는 속도로 제공한다고 소개한다. 앞서 나온 5.3 스파크는 연구 프리뷰였을 뿐 최종 모델이 아니었고, 그 방향을 최신 모델인 5.6 Sol로 가져왔다는 것이다. 처음에는 접근이 제한되지만 용량을 늘려 더 많은 사용자에게 제공할 계획이라고 덧붙인다.

Codex로 크롬 익스텐션 즉석 제작

이어 Codex의 개선점을 데모하는데, 유튜브 스튜디오 라이브 채팅에 올라오는 한국어를 영어로 실시간 번역해 주는 크롬 익스텐션을 즉석에서 만든다. 가브리엘 추아는 ‘플랜 모드’를 굳이 쓸 필요 없이 사람에게 말하듯 Codex와 자연스럽게 대화하면 되며, 계획을 다듬도록 궁금한 것을 물어봐 달라고 하면 플랜 모드보다 더 길고 자연스러운 질문을 던진다고 설명한다. 실행 속도는 medium·fast 등으로 조절할 수 있고 오래 걸리는 작업은 맡겨두고 자리를 비워도 된다. 특히 ‘자동 리뷰’는 모든 명령을 일일이 승인하는 방식과 전부 자동으로 실행하는 ‘YOLO 모드’의 중간 지점으로, 서브 에이전트를 하나 생성해 검토를 맡긴다. 또 Codex에 스스로 목표를 정하게 하는 ‘Goal’ 기능, 작업 중인 에이전트를 방해하지 않고 옆에서 따로 대화하는 ‘사이드 챗’(사이드 챗에서 말하면 Codex가 에이전트가 이해하기 쉬운 말로 정리해 전달), 진행 상황을 물어보기, 스레드 이름 변경·이모지 추가·핀 고정 같은 기능을 보여준다.

개발 편의 기능: API 키 자동화·스크린샷·이미지·데이터 분석

Codex의 편의 기능도 시연한다. ‘OpenAI Developers’ 플러그인을 쓰면 예전처럼 웹사이트에서 직접 API 키를 만들고 다운로드·저장할 필요 없이 키 발급과 로테이트가 자동으로 되고 에이전트·챗GPT 앱 제작 모범사례도 알려주며, 최신 API 문서를 익힌 상태로 바로 만들 수 있다. 스크린샷 기능(커맨드 키, 앱샷)은 겉보기엔 화면 캡처 같지만 실제로는 원본 텍스트와 데이터까지 그대로 담겨 있어, 받은 이메일을 캡처해 답장을 요청하는 식으로 활용할 수 있다. 이미지도 생성해 특정 이미지를 만화책 스케치 스타일로 바꾸거나 오픈소스 레포에서 영감받은 ‘이케아 스타일 설명서’를 만들고, 18MB 크기의 ICML 논문 CSV 데이터셋을 받아 초록을 기준으로 분류하고 논문 트렌드를 보는 프레젠테이션까지 만들어 낸다. 화면에는 진행 상황을 보여주는 ‘펫’을 띄우고 페이지를 옮겨도 유지되며, 라이브 코딩을 시키며 함께 놀 수도 있다.

휴대폰·클라우드 연동과 Computer Use, 마무리

마지막으로 어디서든 작업할 수 있도록 스마트폰과 Codex를 연동해, 휴대폰에서 새 대화를 만들어 화면을 공유하며 진행 중인 프로젝트를 확인하거나 새 작업을 시키며 왔다 갔다 할 수 있음을 보여준다. 노트북을 들고 다닐 필요 없이 Codex를 클라우드에서 돌릴 수도 있는데, 디지털오션 플러그인으로 원격 인스턴스(드롭릿)를 만들면 그 안에서 Codex를 돌리는 가상 머신이 되고 SSH 키 발급도 한 번에 되며, 임시 서버라 나중에 내릴 수 있다. ‘하트비트’ 자동화로 5분마다 인프라 준비 여부를 확인하게 하거나 이메일·Slack 처리, 피드백 자동 요약, PR 자동 생성 같은 자동화를 만들 수 있고, 만들어진 파워포인트는 Codex 안에서 직접 편집하고 브라우저를 열 필요 없이 상대에게 바로 전송할 수 있다. 이날 다루지 않은 ‘Computer Use’는 브라우저와 계산기 같은 앱까지 백그라운드에서 제어하며 권한을 물은 뒤 실행한다. 결국 리허설 없이 즉석에서 크롬 익스텐션을 만들어 한국어 채팅을 실시간 번역하는 데 성공했는데, 이 모든 것이 아직 5.5로 구현된 것이고 5.6은 훨씬 더 좋아질 것이라는 말로 마무리한다.
